# MeuFenil
MeuFenil é um web app focado no controle diário da ingestão de fenilalanina para pessoas com fenilcetonúria (PKU).
O projeto permite registrar alimentos, acompanhar limites diários, visualizar histórico e estatísticas, favoritar alimentos mais consumidos e gerenciar exames laboratoriais de PKU em mg/dL.

O sistema foi pensado para uso prático no dia a dia, com foco em simplicidade, experiência mobile e confiabilidade dos dados.
Também oferece recursos de administração da dieta por terceiros mediante autorização explícita do usuário, atendendo cenários comuns de acompanhamento familiar ou clínico.

Este é um projeto sem fins lucrativos, criado para contribuir com a qualidade de vida de pacientes com PKU no Brasil, oferecendo uma ferramenta acessível, transparente e baseada em informações públicas e personalizáveis.
